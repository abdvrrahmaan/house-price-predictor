# 🏠 House Price Prediction using Linear Regression

## 📌 Project Overview
This project predicts house prices using Machine Learning (Linear Regression).  
It uses features like area, rooms, and other house attributes.

---

## 📊 Dataset
Dataset used: Ames Housing / Kaggle House Price Dataset  
Contains features like:
- Lot Area
- Number of Rooms
- Year Built
- Overall Quality
- Sale Price (Target)

---

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

---

## 🧠 Model Used
- Linear Regression (Supervised Learning)

---

## 📈 Performance
- MAE: ~20,000
- RMSE: ~33,000
- R² Score: ~0.85
